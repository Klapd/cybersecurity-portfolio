Linux SSH Brute Force Detection and Triage Using Splunk

Completed: May 20, 2026

The lab included:

  • Splunk Enterprise deployment in Docker
  • Linux authentication log ingestion
  • SSH brute force attack simulation
  • Custom SPL detection queries
  • Real-time alert configuration
  • Security event triage and analysis
  • SOC-style investigation workflow

Windows Server Event Log Forwarding Using Splunk

Completed: May 27, 2026

This project demonstrates centralized Windows event log collection from a Windows Server 2022 virtual machine into Splunk Enterprise using Splunk Universal Forwarder. The lab focused on forwarding Windows security, system, and application logs into a SIEM environment for visibility and analysis.

The lab included:

  • Splunk Universal Forwarder installation on Windows Server 2022
  • Receiving indexer configuration using TCP port 9997
  • Windows Event Log collection through inputs.conf
  • Security, System, and Application log forwarding
  • Splunk forwarder service restart and validation
  • Windows event log ingestion verification in Splunk Enterprise
  • Foundational SIEM and Windows monitoring workflow
